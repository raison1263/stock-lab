# 📈 내 증권 — 폰에서 보는 개인용 주식 분석 (토스 스타일 다크)

한국 + 미국 주식을 검색·관심목록으로 보고, 시장 순위(발견)와 상세 분석
(차트·종목정보·백테스트·예측)까지. 갤럭시 등 어떤 폰에서도 브라우저로 접속.

---

## 1. 내 컴퓨터에서 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```
브라우저가 열리면 위쪽 검색창에 "삼성전자" 또는 "AAPL"을 입력해 보세요.

## 2. 📱 폰에서 어디서나 (무료, Streamlit Community Cloud)
1. GitHub 저장소에 아래 파일들을 올립니다.
2. share.streamlit.io → GitHub 로그인 → Create app → 저장소·`main`·`app.py` 선택 → Deploy.
3. 생긴 `https://...streamlit.app` 주소를 폰 크롬으로 열고 ⋮ → "홈 화면에 추가".

### 나만 보게 잠그기(선택)
배포한 앱 Settings → Secrets 에 `app_password = "원하는비번"` 한 줄 추가.

## 3. 파일 구성 (이 5개를 GitHub에 올리면 됨)
- `app.py` — 화면(관심·발견·상세)
- `stock_lab.py` — 분석 엔진(데이터·지표·전략·백테스트·예측·순위)
- `.streamlit/config.toml` — 다크 테마 (반드시 `.streamlit` 폴더 안에)
- `requirements.txt` — 의존성
- `README.md` — 이 안내서

## 4. 화면 구성
- **⭐ 관심** — 내 관심 종목을 오늘 등락률 순으로. 종목 탭 → 상세.
- **🔍 발견** — 상단 지수 바(코스피·나스닥·S&P500 등) + 국내/해외/전체 필터 +
  거래대금·거래량·급상승·급하락 순위(주요 90여 종목 풀 기준).
- **상세** — 큰 가격 + 4개 탭:
  - 📈 차트: 캔들 + 거래량(상승 빨강·하락 파랑), 기간 1주~5년
  - 📋 종목정보: 1일·1년 시세 범위, 시작가/종가/거래량/거래대금, 일별 시세 표
  - 🧪 백테스트: 전략(결합 가능)+손절/익절, 단순보유 대비 성과
  - 🤖 예측: 확률 부채꼴(1주~5년) + (3개월 이하)방향 상승확률

## 5. 종목 코드
- 한국: 6자리 — 삼성전자 `005930`, SK하이닉스 `000660`, 카카오 `035720`
- 미국: 심볼 — `AAPL`, `MSFT`, `NVDA`

## ⚠️ 한계와 면책
- 실시간 체결·호가·투자자 동향·매매 기능은 없습니다(분석 도구).
- 예측은 '확률 범위'이며 뉴스·실적 충격을 반영하지 못합니다. 기간이 길수록
  범위가 넓어지는 게 정상입니다.
- 백테스트 성과는 미래를 보장하지 않으며, 모든 투자 판단과 결과의 책임은 본인에게 있습니다.
- 발견 화면 첫 로딩/필터 변경 시 종목 시세를 받느라 잠깐(10~20초) 걸릴 수 있고,
  이후엔 캐시되어 빨라집니다.
