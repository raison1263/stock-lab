# 📈 내 증권 — 폰에서 보는 개인용 주식 분석 (토스 증권 스타일)

한국 + 미국 주식을 검색해서 현재가·차트를 보고, 전략 백테스트·신호 스크리닝·
머신러닝 상승확률까지 한 화면에서. 갤럭시 S26 울트라 등 어떤 폰에서도 브라우저로 접속.

---

## 1. 내 컴퓨터에서 먼저 실행해 보기
```bash
pip install -r requirements.txt
streamlit run app.py
```
브라우저가 열리면 위쪽 검색창에 "삼성전자" 또는 "AAPL"을 입력해 보세요.

> 첫 실행 때 종목 목록을 한 번 내려받느라 몇 초 걸리고, 이후엔 캐시되어 빨라집니다.

---

## 2. 📱 폰에서 어디서나 보기 (무료, 약 10분)

**Streamlit Community Cloud**에 올리면 인터넷 주소(URL)가 생기고, 폰 브라우저로 어디서든 접속할 수 있어요.

1. **GitHub에 올리기** — github.com 가입 후 새 저장소(repository)를 만들고
   `app.py`, `stock_lab.py`, `requirements.txt`, `.streamlit/config.toml` 4개를 올립니다.
   (저장소는 **비공개(private)** 로 둬도 됩니다.)
2. **배포** — share.streamlit.io 접속 → GitHub로 로그인 → "New app" →
   방금 만든 저장소와 `app.py`를 선택 → Deploy.
3. 잠시 뒤 `https://...streamlit.app` 주소가 생깁니다. 이게 내 앱이에요.

### 갤럭시에서 앱처럼 쓰기
폰 크롬으로 그 주소를 연 뒤, 우상단 ⋮ 메뉴 → **"홈 화면에 추가"** 를 누르면
홈 화면에 아이콘이 생겨 일반 앱처럼 바로 열 수 있어요.

> 무료 플랜은 한동안 안 쓰면 잠들었다가, 다시 열면 30초쯤 뒤 깨어납니다.

---

## 3. 🔒 나만 보게 잠그기 (선택)

공개 URL이라도 비밀번호로 막을 수 있어요. Streamlit Cloud 앱 화면에서
**Settings → Secrets** 에 아래 한 줄을 넣고 저장하면 됩니다.
```toml
app_password = "원하는_비밀번호"
```
이러면 접속 시 비밀번호를 물어보고, 맞아야 화면이 열립니다.
(비밀번호를 설정하지 않으면 잠금 없이 바로 열립니다.)
더 강하게 막으려면 앱 Settings에서 **내 구글 계정만 볼 수 있도록** 뷰어를 제한할 수도 있어요.

## 4. 화면 구성
- **상단 검색** — 이름/코드로 종목 검색 후 탭하면 전환
- **현재가 카드** — 종가·전일대비 (한국식: 상승 빨강 🔴 / 하락 파랑 🔵)
- **📈 차트** — 기간 버튼(1주~5년)이 있는 토스풍 라인차트 + 핵심 지표
- **🧪 백테스트** — 전략(결합 가능)+손절/익절 적용, 매매시점·성과
- **🔍 신호** — 관심 종목의 오늘 신호(골든크로스·과매도 등)를 배지로 표시
- **🤖 예측** — 기간(1주~5년) 선택, 확률 부채꼴 + (3개월 이하)방향 상승확률

## ⚠️ 예측의 한계 (꼭 읽어주세요)
예측 탭에서는 기간을 1주부터 5년까지 고를 수 있습니다. 하지만 미래는
한 점(가격)으로 맞힐 수 없어 **'확률 범위(부채꼴)'** 로 보여줍니다. 기간이 길수록
범위가 넓어지는 게 정상이고, 그 넓이 자체가 가장 정직한 정보예요.

- **변동성만 보기(추세 제거)** 토글: 최근 추세를 5년까지 곱하면 비현실적이라,
  1년 이상에선 추세를 빼고 변동성만 보는 게 기본으로 켜집니다.
- **방향 상승확률 모델**: 3개월 이하에서만 제공합니다. 그보다 먼 미래는 검증할
  독립 표본이 거의 없어 정확도 자체가 의미가 없어서 부채꼴 범위만 보여줍니다.
- **뉴스·실적 충격은 전혀 반영하지 못합니다.**

이 도구는 분석·학습용이고, 사고팔지의 최종 판단과 그 결과의 책임은 본인에게 있습니다.
